<a name=8.0></a>8.0 changes
-----
* Window functions,
* Common Table Expression
* DDL
* groupng()
* regular expressions
* group replication
* invisible indexes
* undo log - no longer in tablespace
* persist variables
* mandatory roles
* slowdown on brute force attacks
* auto-upgrade
* upgrade checker
* trandactional data dictionary
* temporary tables use new engine, new parameters
* redo logs
